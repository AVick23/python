Переменные и типы данных

Переменные Типы данных Именование

Что такое переменная?

Переменная — это именованная область памяти, которая хранит данные. В нашем чат-боте мы уже используем переменные для хранения имени пользователя.

# Шаг 2: Добавляем переменные в наш бот
print("Привет! Я чат-бот. Как тебя зовут?")
name = input("Введите ваше имя: ")  # name - переменная типа str

# Добавляем дополнительные переменные
greeting = "Рад знакомству, " + name + "!"  # greeting - тоже строка
is_new_user = True  # is_new_user - булево значение
user_count = 1  # user_count - целое число

print(greeting)
print(f"Вы {user_count} пользователь бота!")

Совет: Имена переменных должны быть осмысленными и отражать их назначение. В Python используется snake_case (слова_разделены_подчеркиванием).

Правила именования переменных

  • Могут содержать буквы, цифры и символ подчеркивания
  • Не могут начинаться с цифры (но могут содержать цифры)
  • Регистрозависимы (myVar и myvar — разные переменные)
  • Не могут совпадать с ключевыми словами Python (if, for, while и т.д.)
  • Используйте snake_case для имен переменных
# Шаг 3: Корректные имена переменных в нашем боте
user_name = input("Введите ваше имя: ")  # правильно
user_count = 1  # правильно

# Неправильные имена в контексте бота
2user = "Иван"  # ошибка: начинается с цифры
user-name = "Иван"  # ошибка: дефис не разрешен
class = "Python"  # ошибка: class - ключевое слово

Основные типы данных в нашем боте

Числовые типы

  • int — целые числа: user_count = 1
  • float — числа с плавающей точкой: bot_version = 1.0

Строковые и другие типы

  • str — строки: user_name = "Иван"
  • bool — логические значения: is_new_user = True
  • NoneType — отсутствие значения: user_preferences = None

Динамическая типизация в действии

В Python переменная может менять тип в процессе выполнения программы. Но в нашем боте мы будем стараться сохранять смысловую целостность переменных.

# Шаг 5: Избегаем ненужных изменений типов в боте
user_count = 1  # int
print(f"Тип user_count: {type(user_count)}")  # 

# В реальном боте лучше НЕ делать так:
# user_count = "один"  # теперь это str
# Это приведет к ошибкам при попытке увеличить счетчик

# Правильный подход - использовать отдельные переменные
user_count = 1
user_count_str = "один"

print(f"Теперь user_count: {user_count} (тип {type(user_count)})")
print(f"А user_count_str: {user_count_str} (тип {type(user_count_str)})")

Важно: Хотя Python позволяет менять тип переменной, это может привести к неочевидным ошибкам. Старайтесь сохранять смысловую целостность переменных — если переменная предназначена для хранения числа, не превращайте её в строку без необходимости.

Приведение типов

Преобразование Типы данных Ввод/вывод

Что такое приведение типов в нашем боте?

Приведение типов — это процесс преобразования значения из одного типа данных в другой. В нашем чат-боте это особенно важно при работе с пользовательским вводом.

Явное приведение в боте

Когда нам нужно преобразовать строку в число:

age_str = input("Сколько вам лет? ")
age = int(age_str)  # строка в число

Неявное приведение в боте

Когда Python автоматически преобразует типы:

total = 1 + 0.5  # int автоматически преобразуется в float
print(type(total))  #

Основные функции приведения типов в боте

Числа в боте

  • int() — преобразует в целое число для счетчиков
  • float() — преобразует в число с плавающей точкой для версий

Строки в боте

  • str() — преобразует в строку для вывода
  • bool() — преобразует в булево значение для проверок
# Шаг 6: Добавляем обработку возраста в наш бот
user_name = input("Как вас зовут? ")
print(f"Привет, {user_name}!")

# Преобразование строки в число
age_str = input("Сколько вам лет? ")
age = int(age_str)  # преобразуем строку в целое число

# Используем f-строку для вывода
print(f"Вам {age} лет. Через год вам будет {age + 1} лет.")

# Преобразование числа в строку для конкатенации
print("Вам " + str(age) + " лет.")

Практическое применение в чат-боте

В нашем чат-боте приведение типов особенно важно при работе с пользовательским вводом:

# Шаг 7: Безопасное получение возраста с обработкой ошибок
def get_age():
    """Безопасно получает возраст от пользователя"""
    while True:
        age_str = input("Сколько вам лет? ")
        try:
            age = int(age_str)  # преобразуем строку в число
            if age < 0:
                print("Возраст не может быть отрицательным. Попробуйте еще раз.")
                continue
            return age
        except ValueError:
            print("Пожалуйста, введите число.")

# Используем в основном коде бота
user_name = input("Как вас зовут? ")
print(f"Привет, {user_name}!")

age = get_age()
if age < 18:
    print("Вы еще несовершеннолетний.")
else:
    print("Вы совершеннолетний.")

Важно: При преобразовании строки в число используйте обработку исключений (try/except), так как пользователь может ввести нечисловое значение, что вызовет ошибку ValueError.

Частые ошибки при приведении типов в боте

  • Попытка преобразовать нечисловую строку в число: int("text") → ValueError
  • Забыть, что input() всегда возвращает строку
  • Смешивать разные типы в операциях без преобразования: "Возраст: " + 25 → TypeError
  • Неправильно обрабатывать десятичные числа: int("3.14") → ValueError

Решение: Всегда проверяйте тип данных перед операцией. Для конкатенации строк и чисел используйте преобразование в строку: "Возраст: " + str(25) или f-строки: f"Возраст: {25}"

# Шаг 11: Добавляем обработку ошибок и расширяем функционал
import random
import requests

API_KEY = "4541720618481add2ebd65f107f05112"
BASE_URL = "http://api.openweathermap.org/data/2.5/weather"

def get_weather(city):
    """Получает погоду для указанного города"""
    params = {
        "q": city,
        "appid": API_KEY,
        "units": "metric",
        "lang": "ru"
    }
    
    try:
        response = requests.get(BASE_URL, params=params)
        response.raise_for_status()  # Вызовет исключение для кодов ошибок HTTP

        temp = data["main"]["temp"]
        feels_like = data["main"]["feels_like"]

        humidity = data["main"]["humidity"]

        return f"""🏙️ Погода в городе {city.capitalize()}:

☁️  Состояние: {description.capitalize()}
💧 Влажность: {humidity}%
💨 Скорость ветра: {wind_speed} м/с"""
    
    except requests.exceptions.RequestException as e:
        return f"Ошибка при получении погоды: {str(e)}"

        return f"Ошибка обработки данных: {str(e)}"

            return "Вы еще несовершеннолетний."
        else:
            return "Вы совершеннолетний."
    else:
        return "Извините, я не понял ваш запрос. Могу помочь с приветствием, узнать как у вас дела или попрощаться."

def main():
    """Основная функция программы"""
    print("Загрузка чат-бота...")
    
    name = input("Как вас зовут? ").strip()
    if not name:
        name = "Пользователь"
    
    print(greet(name))
    
    while True:
        user_input = input("\nВы: ")
        
        if not user_input.strip():
            print("Бот: Пожалуйста, введите сообщение.")
            continue
            
        response = process_message(user_input, name)
        print("Бот:", response)
        
        # Выход при получении команды "пока"
        if "пока" in user_input.lower():
            break

if __name__ == "__main__":
    main()

Частые ошибки новичков

Отступы

В Python они значимы. Используйте 4 пробела, не смешивайте с табами.

🔄

Версии

Команда python может указывать на разные версии. Всегда проверяйте python --version.

🔠

Кодировка

Python 3 работает с Unicode по умолчанию — печатайте строки как обычный текст.

📦

Среда

Путаница глобальных пакетов. Работайте в venv, чтобы зависимости не конфликтовали.

Эти ошибки легко избежать, если знать о них заранее. Python дает понятные сообщения об ошибках — читайте их внимательно!
