Ввод/вывод Переменные Строки

1. Ввод и вывод: "Голос" вашего бота

# Пример 1: Приветствие
print("Привет! Я чат-бот. Как тебя зовут?")
name = input("Введите ваше имя: ")
print("Рад знакомству, " + name + "!")

Что изучаем:

  • print() — вывод текста в терминал
  • input() — получение данных от пользователя
  • Конкатенация строк (объединение через +)
  • Знакомство с переменными (хранение информации)

Важно отметить:

  • В Python 3 print — это функция, поэтому нужны скобки
  • input() всегда возвращает строку, даже если введено число
  • Отступы не нужны для простых команд (мы еще вернемся к ним)
Условия Сравнение Отступы

2. Условные операторы: "Мозг" вашего бота

# Пример 2: Персонализированный ответ
print("Привет! Я чат-бот. Как тебя зовут?")
name = input("Введите ваше имя: ")

if name == "Админ":
    print("Добро пожаловать, создатель!")
elif name == "Иван":
    print("Здравствуйте, Иван! Рад вас видеть снова.")
else:
    print("Привет, " + name + "! Рад знакомству.")

Что изучаем:

  • Условные операторы if/elif/else
  • Оператор сравнения ==
  • Отступы (обязательны в Python для определения блоков кода)
  • Логические выражения

Важно отметить:

  • В Python отступы — часть синтаксиса (обычно 4 пробела)
  • == для сравнения, = для присваивания
  • Строки сравниваются с учетом регистра ("Иван" ≠ "иван")
Циклы Методы строк Ключевые слова

3. Циклы: "Повторение" в диалоге

# Пример 3: Диалог с возможностью выхода
print("Привет! Я чат-бот. Как тебя зовут?")
name = input("Введите ваше имя: ")

while True:
    message = input("Вы: ")
    
    if message.lower() == "привет":
        print("Бот: Здравствуйте, " + name + "!")
    elif message.lower() == "как дела?":
        print("Бот: У меня все отлично, спасибо что спросили!")
    elif message.lower() == "пока":
        print("Бот: До свидания, " + name + "!")
        break
    else:
        print("Бот: Извините, я не понял ваш запрос.")

Что изучаем:

  • Цикл while для постоянного взаимодействия
  • Метод .lower() для преобразования строки в нижний регистр
  • Ключевое слово break для выхода из цикла
  • Продолжаем работать с условиями

Важно отметить:

  • Бесконечный цикл while True требует условия выхода
  • .lower() помогает сделать обработку запросов регистронезависимой
  • Отступы определяют, какой код выполняется внутри цикла
Функции Документирование Структура кода

4. Функции: "Модули" вашего бота

# Пример 4: Использование функций
def greet(name):
    """Приветствует пользователя по имени"""
    return "Привет, " + name + "! Я чат-бот."

def process_message(message, name):
    """Обрабатывает сообщение пользователя"""
    message = message.lower()
    
    if message == "привет":
        return "Здравствуйте, " + name + "!"
    elif message == "как дела?":
        return "У меня все отлично, спасибо что спросили!"
    elif message == "пока":
        return "До свидания, " + name + "!"
    else:
        return "Извините, я не понял ваш запрос."

# Основной код
name = input("Как вас зовут? ")
print(greet(name))

while True:
    user_input = input("Вы: ")
    
    response = process_message(user_input, name)
    print("Бот:", response)
    
    if user_input.lower() == "пока":
        break

Что изучаем:

  • Объявление функций через def
  • Параметры и возвращаемые значения
  • Документирование функций (docstrings)
  • Повторное использование кода

Важно отметить:

  • Функции помогают структурировать код
  • Имена функций должны быть глаголами (что делает функция)
  • Докстринги — это не комментарии, а документация, которая может быть использована инструментами
Исключения Типы данных f-строки

5. Обработка ошибок: "Защита" вашего бота

# Пример 5: Добавление обработки ошибок
def get_age():
    """Безопасно получает возраст от пользователя"""
    while True:
        try:
            age = int(input("Сколько вам лет? "))
            if age < 0:
                print("Возраст не может быть отрицательным. Попробуйте еще раз.")
                continue
            return age
        except ValueError:
            print("Пожалуйста, введите число.")

# Использование в основном коде
name = input("Как вас зовут? ")
print(f"Привет, {name}!")

age = get_age()
if age < 18:
    print("Вы еще несовершеннолетний.")
else:
    print("Вы совершеннолетний.")

Что изучаем:

  • Блок try/except для обработки исключений
  • Преобразование типов (int())
  • Циклы внутри функций для повторного запроса
  • f-строки для форматирования

Важно отметить:

  • Не все вводы пользователя можно предсказать
  • ValueError возникает при попытке преобразовать нечисловой ввод в число
  • f-строки (Python 3.6+) — удобный способ форматирования строк
Пример

Полный пример чат-бота для студентов

import random
import requests


API_KEY = "4541720618481add2ebd65f107f05112"
BASE_URL = "http://api.openweathermap.org/data/2.5/weather"
   
def greet(name):
    """Приветствует пользователя по имени"""
    greetings = [
        f"Привет, {name}! Рад тебя видеть!",
        f"Здравствуйте, {name}! Как я могу вам помочь?",
        f"Добро пожаловать, {name}! Я ваш персональный бот-помощник."
    ]
    return random.choice(greetings)

def weather():
    print("\nКруто, вы хотите узнать погоду")
    city = input("\nВведите название города: ")


    params = {
    "q": city,
    "appid": API_KEY,
    "units": "metric",  
    "lang": "ru"        
    }


    response = requests.get(BASE_URL, params=params)


    if response.status_code == 200:

        data = response.json()


        temp = data["main"]["temp"]
        feels_like = data["main"]["feels_like"]
        description = data["weather"][0]["description"]
        humidity = data["main"]["humidity"]
        wind_speed = data["wind"]["speed"]


    message = f"""🏙️ Погода в городе {city.capitalize()}:
    🌡️  Температура: {temp}°C (ощущается как {feels_like}°C)
    ☁️  Состояние: {description.capitalize()}
    💧 Влажность: {humidity}%
    💨 Скорость ветра: {wind_speed} м/с"""
    return message

def get_age():
    """Безопасно получает возраст от пользователя"""
    while True:
        try:
            age = int(input("Сколько вам лет? "))
            if age < 0:
                print("Возраст не может быть отрицательным. Попробуйте еще раз.")
                continue
            return age
        except ValueError:
            print("Пожалуйста, введите число.")
    

def process_message(message, name):
    """Обрабатывает сообщение пользователя"""
    message = message.lower().strip()
    
    if "привет" in message or "здравствуй" in message:
        return "Здравствуйте! Как я могу вам помочь?"
    elif "как дела" in message:
        return "У меня все отлично, спасибо что спросили! А у вас как?"
    elif "погода" in message:
        weather()
    elif "пока" in message:
        print(f"До свидания, {name}! Приходите еще.")
        exit()
    elif "спасибо" in message:
        return "Всегда пожалуйста! Рад был помочь."
    elif "возраст" in message:
        message = get_age()
        if message < 18:
            return "Вы еще несовершеннолетний."
        else:
            return "Вы совершеннолетний."
    else:
        return "Извините, я не понял ваш запрос. Могу помочь с приветствием, узнать как у вас дела или попрощаться."

def main():
    """Основная функция программы"""
    print("Загрузка чат-бота...")
    
    name = input("Как вас зовут? ").strip()
    if not name:
        name = "Пользователь"
    
    print(greet(name))
    
    while True:
        user_input = input("\nВы: ")
        
        if not user_input.strip():
            print("Бот: Пожалуйста, введите сообщение.")
            continue
            
        response = process_message(user_input, name)
        print("Бот:", response)
        

if __name__ == "__main__":
    main()

Обратите внимание на структуру кода: каждая функция решает одну конкретную задачу, что делает код понятным и легко модифицируемым.

Советы

Частые ошибки новичков

↹

Отступы

В Python они значимы. Используйте 4 пробела, не смешивайте с табами.

🔄

Версии

Команда python может указывать на разные версии. Всегда проверяйте python --version.

🔠

Кодировка

Python 3 работает с Unicode по умолчанию — печатайте строки как обычный текст.

📦

Среда

Путаница глобальных пакетов. Работайте в venv, чтобы зависимости не конфликтовали.

Эти ошибки легко избежать, если знать о них заранее. Python дает понятные сообщения об ошибках — читайте их внимательно!
